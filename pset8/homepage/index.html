Hyper Text Markup Language (HTML) is the standard markup language used to create web pages. HTML documents consist of a series of elements enclosed in angle brackets < >, which are known as HTML tags. These tags are used to define the structure and content of a web page. For example, in these 3 accordian style collapsible boxes. HTML tags button, div, strong and h2 are used.

Cascading Style Sheets (CSS) is a style sheet language used in web development to control the visual presentation and layout of HTML documents. CSS allows web designers and developers to style elements on a web page including attributes such as colors, fonts, spacing, positioning, and more. JavaScript (JS) is a high-level, versatile, and dynamically typed programming language primarily known for its use in web development. It allows developers to add interactivity, manipulate the Document Object Model (DOM), and create dynamic content on websites. JavaScript can be used both in web browsers and on server-side platforms.
